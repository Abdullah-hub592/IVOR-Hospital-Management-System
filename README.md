# IVOR Hospital Management System

## Overview

IVOR Hospital Management System is a Database Management System (DBMS) project developed using SQL, PHP, HTML, and CSS. The system is designed to efficiently manage hospital operations, including patient records, doctor information, wards, consultant teams, treatments, complaints, and staff management through an interactive web-based interface.

## Features

### Dashboard

The system provides a dynamic dashboard displaying real-time statistics including:

* Total Patients
* Total Wards
* Total Nurses
* Total Beds
* Total Doctors
* Total Specialties

### Record Management

The system allows users to manage:

* Patient Records
* Ward Records
* Consultant Team Records
* Doctor Information
* Nurse Information
* Bed Allocation
* Specialties

### CRUD Operations

Users can:

* Add new patients
* Add doctors
* Add wards
* Add nurses
* Manage consultant teams
* Update records
* View stored information

## Technologies Used

* SQL (Database Design & Queries)
* PHP (Backend Development)
* HTML (Frontend Structure)
* CSS (User Interface Design)

## Implemented SQL Queries

1. List of consultants and the doctors in their team.
2. List of wards with respective sisters, care units, and staff nurses in charge of care units.
3. List of patients along with their complaints, treatments, and treatment dates.
4. List of junior housemen, their patients, and the staff nurse responsible for the patient's care unit.
5. List of consultants with a unique specialty.
6. List of complaints, treatments provided, and the experience history of doctors performing those treatments.
7. List of patients having more than one complaint and their treatments.
8. List of patients grouped by treatment within complaints.
9. Performance history of a particular doctor.
10. Complete medical details of a particular patient.
11. List of treatments given for a specific complaint between two dates, ordered by treatment.
12. List of staff positions in the hospital along with the count of employees in each position.

## Objectives

* Demonstrate relational database design concepts.
* Implement SQL queries for complex hospital data retrieval.
* Provide an easy-to-use interface for hospital management.
* Integrate frontend and backend technologies with a structured database system.
